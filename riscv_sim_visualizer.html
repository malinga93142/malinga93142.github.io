Formulas

CPI = Cycles / Instructions_Retired IPC = Instructions_Retired / Cycles CPI = 1 / IPC Cycles = Instructions_Retired × CPI
This render uses a Ripes generated JSON with flags enabled (RISC-V pipeline sim).

Pipeline stages:
  1. IF — Instruction Fetch
  2. ID — Instruction Decode
  3. II — Second Issue (6S dual-issue processors only)
  4. EX — Execute
  5. MEM — Memory Access
  6. WB — Write Back

Stalls (---) indicate pipeline hazards — data or control dependencies that prevent forward progress.
CPI > 1 implies stalls or structural hazards; ideal CPI = 1 (one instruction retired per cycle).
IPC = 1 / CPI. A fully pipelined, hazard-free execution achieves IPC = 1 on a 5-stage in-order processor.
Processor: RV32 5-stage in-order pipeline, no forwarding (NO_FW) — all hazards resolved by stalling.
