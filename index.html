About Me

Hi! I'm Malinga RK (MARK) — a passionate computer science student with a deep interest in systems programming, debugging tools, and low-level software design. I enjoy exploring how computers work under the hood — from ELF internals and process tracing to performance analysis and compiler toolchains.

My goal is to build efficient, reliable software that interacts closely with the operating system and hardware. I'm currently learning about memory management, binary instrumentation, and OS internals.

This site is where I'll document my journey — including tools I build, articles I write, and experiments I conduct in system-level software.
